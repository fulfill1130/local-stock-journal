# Web Prototype Track

The Web Prototype Track is the current Flask app.

It is:

- A feature lab.
- A validation surface.
- The current place to prove accounting, import staging, market data, ETF holdings, provider, and corporate action workflows.
- Useful for testing workflows before concepts are promoted to the desktop product.

It is not:

- The final desktop UI.
- A polished consumer app shell.
- A desktop packaging target.
- A reason to preserve web/mobile layout decisions in the final desktop product.
